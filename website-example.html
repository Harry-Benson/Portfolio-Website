Project 1

This is a sample of my website designing skill

I designed this website with the marquees specifically in mind, as when wireframing I thought of the idea and I knew I had to make it happen. So, it was the first thing I worked on and what I modeled almost the entire website around, starting with the marquees then adding a body, header and footer from there. I then ran into some trouble because I had designed the marquee to move images, but needed instead to move <div> elements as they were much easier to add text and anchor elements to, so I had to restructure that entire section of the website, which was difficult but once it was done I was able to add a background image, text and an anchor element that brought you to this page! To make the website, I used VScode for the coding and github desktop to store and keep my changes in a repository, and used both html and css to make my website function, with html for the structuring and css for the layout and visuals. In the future, I would like to improve my website in many ways, some of which being bugfixing, such as when resizing the window vertically, the wave graphics on either side of the page float off of the edge, as well as adding mobile functionality by making the marquees switch from the sides to the top and bottom. I would also like to improve on the visuals, adding more to the body of the page to make it pop more, as it is currently quite minimalist, as well as adding reactivity to different parts of the website, not just the marquee, in both hover colours, sizing and maybe even particles when hovering over certain elements. I would also like to add the contact page, as well as making the social links functional, if I ever repurpose this website for an actual portfolio later on, as well as adding other projects that I have worked on. However, for the sake of anonymity and this project, I have not.
